┌────────────────────────────────────────────────────────────────┐
│  CONTROLS (Desktop & Web)                                      │
└────────────────────────────────────────────────────────────────┘

   🎮 KEYBOARD SHORTCUTS:
   
      H      = Holo Mode (turn model blue)
      W      = Wireframe (toggle wireframe view)
      R      = Reset view
      A      = Auto-rotate
      L      = Load model
      ESC    = Exit / Close model list

   🖱️  MOUSE CONTROLS:
   
      Left Drag     = Rotate model
      Right Drag    = Pan camera
      Scroll        = Zoom in/out

   🎙️  VOICE COMMANDS:
   
      Click "Voice Control" button, then say:
      - "Holo mode"
      - "Wireframe"  
      - "Auto rotate"
      - "Reset"
      - "Load [model name]"

┌────────────────────────────────────────────────────────────────┐
│  ADDING YOUR OWN 3D MODELS                                     │
└────────────────────────────────────────────────────────────────┘

   1. Create a folder in "3D Models\" directory
   2. Name it whatever you want (e.g., "My Model")
   3. Put your .obj file inside
   4. (Optional) Add .mtl and texture files
   5. Click "Load" in HoloView to see it!

   Supported: .obj files with optional .mtl and textures

┌────────────────────────────────────────────────────────────────┐
│  TROUBLESHOOTING                                               │
└────────────────────────────────────────────────────────────────┘

   ❌ "Python not found"
      → Install Python from python.org
      → Check "Add to PATH" during installation

   ❌ Models not showing
      → Check "3D Models" folder exists
      → Verify .obj files are inside model folders

   ❌ Voice control not working  
      → Normal - requires SpeechToText.py module
      → Works fine without it using keyboard/mouse

   ❌ .exe won't start
      → Run as Administrator
      → Add Windows Defender exception
      → Keep "3D Models" folder with .exe
